# Recommended Resources

## Books

1. **AWS Certified DevOps Engineer Professional Study Guide** – Exam-aligned deep dive with labs
2. **Terraform: Up & Running** by Yevgeniy Brikman – Practical patterns for IaC
3. **Amazon Web Services in Action** – Broad AWS coverage for architects and builders
4. **The DevOps Handbook** – Culture, automation, measurement, and sharing fundamentals
5. **Kubernetes Up & Running** – Operate Kubernetes clusters confidently

## Tools & Utilities

| Category | Tool | Why It Matters |
| --- | --- | --- |
| CLI | [AWS CLI](https://aws.amazon.com/cli/) | Scriptable access to every AWS API |
| Serverless | [AWS SAM CLI](https://aws.amazon.com/serverless/sam/) | Build & test Lambda apps locally |
| Containers | [Docker](https://www.docker.com/) | Build container images for ECS/EKS |
| Kubernetes | [kubectl](https://kubernetes.io/docs/tasks/tools/) | Control EKS clusters |
| IaC | [Terraform](https://www.terraform.io/) | Multi-cloud infrastructure management |
| Observability | [eksctl](https://eksctl.io/), [Helm](https://helm.sh/) | Simplify cluster lifecycle |

### Study Strategy

- Combine a long-form course with project-based labs from this repo.
- Keep a changelog of lessons learned per project.
- Schedule weekly reviews and mock interviews to retain knowledge.
